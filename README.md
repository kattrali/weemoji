## Requirements

## Installation

1. Copy weemoji.py to `~/.weechat/python/autoload`

```bash
mkdir -p ~/.weechat/python/autoload
cd ~/.weechat/python/autoload
wget https://raw.githubusercontent.com/kattrali/weemoji/master/weemoji.py
```

2. Start Weechat
